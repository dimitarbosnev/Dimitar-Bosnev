What is 3D Rendering?

3D Rendering is a course I took during the second year of my CMGT study. It was dedicated to teach us how rendering in 3D space is done using OpenGL. During the course I had to complete 4 assignments which explored shaders, game enigne architecture, rendering, lighing, mesh manipulation and texture blending.

Assignment 1

During this assignment I had to create a fragment shader that displays a checker board pattern and applies a lighting effect based on my mouse position.

Assignment 2

During this assignment I had to create a parent/child system in a micro game engine which was given by the teachers for this course. And implement a camera behaviour which rotates around a pre selected object.

Assignment 3
